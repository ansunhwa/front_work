CSS 선택자

선택자란 ?

특정 thml요소를 선택하고자 할 때 이용하는 기능
해당 요소를 선택하여 원하는 "스타일"과 "기능"을 적용시킬 수 있다

1. 모든 (전체) 선택자 : *

현재 문서의 모든 요소들을 선택하고자 할 때

        *{
            스타일속성 : 값;
            스타일속성 : 값;
            ...
        }

2. 태그 선택자 : 태그명

현재 이 문서에 해당하는 태그들 모두 선택하고자 할 때

        태그명{
            스타일속성 : 값;
            스타일속성 : 값;
            ...
        }

3. 아이디 선택자 : 아이디

특정 html요소 "하나만"을 선택하고자 할 때 사용
단, 해당 요소에 id속성을 이용하여 고유한 아이디값을 부여해야 함 

        #아이디{
            스타일속성 : 값;
            스타일속성 : 값;
            ...
        }

    해당 문서내에 내가 원하는 요소 "여러개"를 선택하고자 할 때 사용 

            .클래스명{
            스타일속성 : 값;
